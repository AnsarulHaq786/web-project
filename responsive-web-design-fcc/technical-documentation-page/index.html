Introduction

C is a general-purpose programming language developed in the early 1970s. It has since become one of the most widely used languages for system and application software.

The C language allows for low-level memory manipulation and is often used in developing operating systems, embedded systems, and high-performance applications.

  • C was created by Dennis Ritchie at Bell Labs.
  • It is known for its efficiency and control over system resources.
// Simple C code #include <stdio.h> int main() { printf("Hello, C!\n"); return 0; }
Hello World

The "Hello, World!" program is traditionally the first program a beginner writes when learning a new language.

It demonstrates the basic structure of a C program, including headers, the `main` function, and output.

  • The `#include <stdio.h>` line includes the standard input-output library.
  • The `printf` function prints the message "Hello, World!" to the console.
// Hello World Program #include <stdio.h> int main() { printf("Hello, World!\n"); return 0; }
Main Function

The `main` function is the entry point of any C program. Execution starts from this function.

In most cases, the `main` function returns an integer value to the operating system, typically `0` for successful execution.

  • Every C program must have a `main` function.
  • It is possible to pass arguments to `main` using command-line arguments.
// Example main function int main() { // Code to be executed return 0; }
Return Statement

The `return` statement is used to exit a function and return a value to the calling function or the operating system.

In the `main` function, returning `0` signifies successful program execution, while other values may indicate different types of errors.

  • The return type of `main` is typically `int` (integer).
  • A non-zero return value can be used to signal an error.
// Return statement example return 0;
Conclusion

C remains one of the most important and widely used programming languages today due to its performance, portability, and control over system resources.

Understanding the basics of C is essential for mastering other languages, as many modern languages derive their syntax and structure from C.

  • C is foundational in learning system-level programming.
  • It is widely used in developing performance-critical applications.
